# 3D Object Display Application

## Description
This application displays 3D objects (cube, triangular pyramid, and triangular prism) on the screen using OpenGL and PyGame. The user can cycle through the models with a key press, displaying one object at a time in the center of the screen.

## Controls
- **SPACE**: Cycle through the 3D objects (cube, pyramid, prism).
- **Close Window**: Quit the application.

## Requirements
- Python 3.x
- PyGame
- PyOpenGL
